Formularios

Un formulario HTML es una sección de un documento que contiene texto, código, elementos especiales llamados controles: casillas de verificación (checkboxes), botones de opción (radio buttons), menúes, etc.; y rótulos (labels) en esos controles.

Los formularios son una forma de enviar datos a un servidor web. Se utiliza la etiqueta contenedora, que incluye todos los campos necesarios para su funcionamiento.

Formularios: atributos action y method

  • action: define el tipo de acción que se llevará a cabo (enviar a un mail o procesar su contenido con un script). También podemos indicar la URL a la que se enviará la petición HTTP con toda la información del formulario.
  • method: define el método de envío de la petición HTTP. Puede ser GET o POST.



Formularios: Atributos

label: se usa para especificar la etiqueta (o nombre) del campo del formulario. Es información para el usuario. Admite el atributo for, que debe tener el mismo valor que el atributo id del campo (input, select o textarea) al que hace referencia la etiqueta.

size: define el tamaño del campo de entrada.

maxlength: define el número máximo de caracteres que se pueden introducir en el campo de entrada.

placeholder: se usa para especificar un texto que se muestra en el campo del formulario hasta que el usuario escriba algo en él. Admite el atributo for, que debe tener el mismo valor que el atributo id del campo (input, select o textarea) al que hace referencia la etiqueta.

input: Atributos del tag input:

  • Type: este valor puede tener muchos valores: text, email, checkbox, color, date, file, hidden, etc. en función del tipo de campo que queramos.

    Formularios: atributo type

    • text: permite ingresar una cadena alfanumérica
    • email: permite ingresar una dirección de correo electrónico
    • checkbox: permite eligir varias opciones
    • color: permite seleccionar un color
    • date: permite seleccionar una fecha
    • file: permite seleccionar un archivo
    • hidden: permite ingresar una cadena alfanumérica
    • month: permite seleccionar un mes
    • number: permite ingresar un número
    • password: permite ingresar una cadena alfanumérica
    • radio: permite elegir una sola opcion
    • range: permite seleccionar un rango
    • search: permite ingresar una cadena alfanumérica
    • time: permite seleccionar una hora
    • url: permite ingresar una dirección web
    • week: permite seleccionar una semana
    • button: button: activa código JavaScript cuando se hace clic en él
  • Name: nombre del campo del formulario.
  • Value: representa el valor que se asignará al campo cuando se envíe la petición HTTP. Permite asignar un valor por defecto al campo.
  • id: este atributo es obligatorio si en el elemento label tiene un atributo for, en tal caso deberá contener un identificador único en la página.

Formularios: fieldset

fieldset: permite agrupar campos de formulario en un bloque.

legend: permite especificar un título para el grupo de campos de formulario.

Formularios: atributos opcionales

  • required: required: el formulario no se podrá enviar hasta que ese campo tenga contenido.
  • disabled: disabled: deshabilita el campo de formulario.
  • readonly: readonly: permite leer el contenido del campo de formulario, pero no permite modificarlo.
  • multiple: multiple: permite seleccionar varias opciones.
  • autofocus: autofocus: activa el campo de formulario cuando se carga la página.
  • value: value: para introducir un valor por defecto en el campo.
